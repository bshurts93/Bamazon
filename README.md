# Bamazon

A simple node.js based marketplace. Shop as a customer or monitor inventory as a manager.

### Prerequisites

- Node.js

## NPM Packages

- MySQL
- Inquirer
- Console.table

### Installing

It's as simple as `npm i`. All dependencies are ready in the `package.json`.
